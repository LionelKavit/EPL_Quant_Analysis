# EPL_Quant_Analysis

The Premier League Tactical Analysis Prompt Flow is designed to automate the tactical evaluation of Premier League teams in the 2023–24 season by integrating real-world performance data with LLM-driven SWOT (Strengths, Weaknesses, Opportunities, Threats) analysis. It empowers coaches, analysts, and sports data enthusiasts to generate dynamic, data-informed tactical insights for any Premier League team in just a few seconds, through Azure AI Studio’s low-code interface.

Within the Prompt Flow, the system accepts two primary inputs: the target Premier League team name and, optionally, the chat history to preserve analytical context across interactions. The Python node processes the filtered team data and computes relevant KPIs, including goals, assists, expected goals, expected assists, and defensive indicators, producing structured summaries for downstream reasoning. These summaries are then passed through a sequence of LLM nodes—Strength, Weakness, Opportunity, and Threat—each powered by GPT-4o via Azure OpenAI. Each node interprets the data through a tactical lens: strengths highlight dominant tactics and key contributors; weaknesses surface vulnerabilities or inefficiencies; opportunities identify growth potential or exploitable matchups; and threats assess risks such as injuries, congestion, or tactical overreliance. A final summary node synthesizes a narrative-style tactical analysis with embedded statistics and contextual reasoning, closely resembling a professional sports journalist’s analytical brief while remaining grounded in deterministic data processing.
